🧮 Scientific Calculator (Java Swing)

A GUI-based Scientific Calculator built using Java Swing.
This application allows users to perform basic arithmetic as well as scientific calculations through an interactive interface.

---

🚀 Features

🔢 Basic Operations

Addition (+)

Subtraction (−)

Multiplication (×)

Division (÷)


📐 Scientific Functions

Square (x²)

Square Root (√)

Trigonometric Functions:

sin()

cos()

tan()



🛠️ Utility Functions

Clear button to reset input

Backspace (⌫) to delete last digit

Input validation with error messages

Prevents division by zero

Handles invalid inputs like √ of negative numbers



---

🖥️ User Interface

 - Built using Java Swing

- Simple and user-friendly layout

- Displays ongoing operation at the top

- Interactive buttons for all inputs



---

🧠 Concepts Used

- Java Swing (GUI)

- Event Handling (ActionListener, Lambda Expressions)

- Mathematical functions using Math class

- Input validation & exception handling

- String manipulation



---

📂 Project Structure

Calculator_Scientific.java


---

▶️ How to Run

1. Make sure Java is installed on your system


2. Compile the program:

javac Calculator_Scientific.java


3. Run the program:

java Calculator_Scientific




---

⚠️ Validations Implemented

- Shows message if input field is empty

- Prevents division by zero

- Prevents square root of negative numbers

- Handles undefined values like tan(90)
